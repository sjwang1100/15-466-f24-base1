How To Play:

The game will randomly show a key in leftmost 3x3 keys (q,w,e,a,s,d,z,x,c) at any time on the 3x3 matrix, but may be at different location. 
ONLY CLICK THE KEY IF IT'S ON THE RIGHT POSITION!!!
